## Configuration

Todoist source connector integrates with the Todoist REST API to extract and synchronize task management
data from your Todoist workspace. It provides comprehensive access to projects, tasks, and
collaboration features with proper hierarchical organization and productivity insights.

### Authentication

This connector uses **OAuth 2.0 authentication**. Connect this source through the Airweave UI, which will guide you through the OAuth flow.

### Configuration Options

This connector does not have any additional configuration options.
